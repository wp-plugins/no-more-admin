== Description ==

This plugin is real simple and basic, it doesn't allow anyone that does not have admin rights to access wp-admin. This will help ensure security.

== Installation ==


1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
